# Sliver Bill Splitter

A modern bill splitting application built with Next.js to scan receipts, extract items, and easily assign costs to different people.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
